# Using-Databases-with-Python

This repository contains clean, well-commented Python solutions for working with SQL databases entirely from Python.

## What you'll find here

- Creation and management of SQL databases from Python  
- Single-table and multi-table database design  
- One-to-many and many-to-many relationships  
- Data import from JSON, XML, CSV, and web sources  
- Complex SELECT queries with JOINs, GROUP BY, and ORDER BY  
- Real-world examples: email organization counts, music library (tracks/artists/albums), course rosters, geographic data  
- Final project with data visualization (interactive maps and charts generated directly from database content)
